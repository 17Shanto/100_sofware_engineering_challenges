# 🧠 Data Structures & Algorithms List

Here are the core computer science concepts, data structures, and algorithmic patterns required to solve the scenarios:

* **Core Data Structures:** Arrays (1D, 2D matrices, Fixed), Strings, Hash Maps, Hash Sets, Linked Lists (Singly and Doubly), Stacks (LIFO), Queues (FIFO, Double-Ended/Deque, Priority Queues, Bounded Queues), and Ring Buffers.
* **Tree & Graph Structures:** Binary Trees, Tries (Prefix Trees), Binary Search Trees, Log-Structured Merge Trees (LSM), Merkle Trees (Hash Trees), Radix Trees, Directed Graphs, and Undirected Graphs.
* **Search & Traversal Algorithms:** Linear Search, Binary Search (including Modified/Rotated variants), Breadth-First Search (BFS), Depth-First Search (DFS), Multi-Source DFS/BFS, Linear Array Sweeps, and K-Way Merge.
* **Sorting & Graph Algorithms:** Stable Sorting, Array Reversal/Cyclic Shifts, Topological Sort (Kahn's), Shortest Path (BFS in Grids), Cycle Detection (Floyd's Tortoise & Hare, Directed Graph Cycle Detection), Lowest Common Ancestor (LCA), and Connected Components (Disjoint Set Union / Flood Fill).
* **Dynamic Programming & Math Concepts:** Tabulation, Coin Change Optimization, Levenshtein Distance Matrix, Manacher's Algorithm, Prefix Sums/Products, Modulo Arithmetic, Combinatorics, and Fibonacci progressions.
* **Algorithmic Patterns:** Two-Pointer Method, Sliding Window Protocol, Expand Around Center, Greedy Traversal, Regular Expressions, and Bit Manipulation / Bitmasks.
* **Distributed Systems & Advanced Architectures:** Consistent Hashing, Cache Eviction Rules (LRU), Operational Transformation (OT), Conflict-Free Replicated Data Types (CRDT), Optimistic/Pessimistic Locking, Two-Phase Commit (2PC), Event Sourcing, Saga Pattern, Raft Consensus Algorithm, Bloom Filters, and Vector Clocks.
